# RheumFellow.com

RheumFellow.com is an open-source rheumatology education framework designed to support fellows, trainees, clinicians, and educators with structured board-style learning, clinical reasoning, and AI-assisted educational workflows.

This public repository contains the open-source educational framework, documentation, templates, and sample materials. The production application, full content library, protected datasets, and premium educational materials are maintained separately.

## Mission

Rheumatology education resources are often fragmented, expensive, and difficult to customize. RheumFellow.com aims to make high-quality rheumatology education more accessible, structured, and reproducible through an open-source, clinician-led framework.

## Goals

- Build reusable templates for rheumatology board-style questions
- Support disease-specific clinical reasoning workflows
- Create transparent educational frameworks for trainees and educators
- Encourage evidence-based, reference-grounded learning
- Provide a foundation for AI-assisted medical education while preserving clinician oversight

## Current Scope

This repository currently includes:

- Educational templates
- Sample board-style question structures
- Documentation for rheumatology learning workflows
- General project roadmap
- Contribution guidelines

## What is not included

To protect intellectual property, patient privacy, and future product development, this public repository does not include:

- Full production application code
- Full proprietary question bank
- Premium educational content
- User data or protected clinical datasets
- API keys, environment files, or deployment secrets
- Internal AI prompts or production workflows

## Intended Audience

- Rheumatology fellows
- Internal medicine residents
- Clinicians preparing for rheumatology boards
- Medical educators
- Clinical researchers interested in AI-assisted education
